# Hackathon

This app provides a form to create to project and an admin panel to view all projects and all participants

## After clone


### Node packages
First of all, open a terminal in the root folder of the project and run
```npm install```. It will read the ```package.json``` file and install all required dependencies.

### Web server
Run ```npm install ng -g``` to install web server. Run ```ng serve``` to start the web server.
The server is listening on ```http://localhost:4200```
